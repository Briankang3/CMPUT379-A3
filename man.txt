NAME
       This is a demostration client-server architecture

SYNOPSIS
       client [ port  ip ] server [port]

DESCRIPTION
       client  sends  requests	for transactions.  server accepts requests and
       process the transactions.

   Arguments
       port   This is the port number that requests will be sent to.

       ip     This is the ip address of the server.

FILES
       source files are client.cpp and server.cpp
